## QUICK START COMMANDS

## FILE PERMISSIONS

```bash
# Make scripts executable
chmod +x scripts/*.sh

# Set proper permissions
chmod 644 .env.example
chmod 600 .env
chmod 644 docker-compose*.yml
chmod 644 backend/requirements*.txt
chmod 644 frontend/package*.json
```

---

## IMPORTANT FILES TO CUSTOMIZE

1. **`.env.example`** → Create `.env` with production credentials
2. **`backend/alembic/env.py`** → Database migration configuration
3. **`frontend/src/config/index.js`** → API URLs and app config
4. **`nginx/nginx.conf`** → Server names and SSL paths
5. **`docker-compose.prod.yml`** → Production environment variables
6. **`postgres/init.sql`** → Database initialization
7. **`monitoring/prometheus/prometheus.yml`** → Metrics configuration

---

# Development setup
cp .env.example .env

### Migration Commands
```bash
# Create new migration
alembic revision --autogenerate -m "description"

# Upgrade to latest
alembic upgrade head

# Downgrade one step
alembic downgrade -1

# View current version
alembic current

# View migration history
alembic history

# Run specific migration
alembic upgrade +1
alembic upgrade <revision_id>
alembic downgrade <revision_id>

# Generate SQL for migration
alembic upgrade head --sql > migration.sql

# Run migration in production
alembic -x db_url=postgresql://prod_user:prod_pass@prod_host:5432/prod_db upgrade head
```

### Monitoring Commands
```bash
# Start monitoring stack
docker-compose -f docker-compose.monitoring.yml up -d

# Check prometheus targets
curl http://localhost:9090/api/v1/targets

# Query prometheus
curl http://localhost:9090/api/v1/query?query=up

# Check alerts
curl http://localhost:9093/api/v2/alerts

# Reload prometheus config
curl -X POST http://localhost:9090/-/reload

# Import grafana dashboards
curl -X POST http://localhost:3001/api/dashboards/db \
  -H "Content-Type: application/json" \
  -d @monitoring/grafana/dashboards/application.json

# View logs
docker-compose -f docker-compose.monitoring.yml logs -f prometheus
docker-compose -f docker-compose.monitoring.yml logs -f grafana
```

### **`Makefile`** - Streamlines development with commands for:

   - Cleanup
